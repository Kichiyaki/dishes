# Dishes

## Development

### Prerequisites

1. Node.JS
2. yarn

### Installation

1. Clone this repo
```
git@github.com:Kichiyaki/dishes.git
```
2. Open the folder with this project in a terminal.
3. ``yarn install``
4. ``yarn run start``
